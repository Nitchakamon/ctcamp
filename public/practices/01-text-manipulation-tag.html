นี่คือตัวอย่างย่อหน้าที่ในไฟล์ HTML มีการเว้นบรรทัดด้วยปุ่ม Enter แต่ว่าเมื่อมาแสดงผล จะไม่มีการเว้นบรรทัด เพราะว่าหากจะให้เว้นบรรทัด ต้องใช้ TAG พิเศษที่เรียกว่า TAG br
เท่านั้น ถึงจะมีการเว้นบรรทัดได้

เพราะว่า Web Browser จะทำการมองการช่องว่างต่างๆ เป็นแค่การเว้นวรรคหนึ่งครั้งเท่านั้น ทดสอบได้ด้วยการ เว้น วรรคยาวๆ ในไฟล์ HTML

และ TAG br ถือเป็น TAG พิเศษ ที่ไม่ต้องมี TAG ปิด เนื่องจาก TAG นี้ไม่ได้เน้นข้อความ แต่เป็น TAG ที่มีไว้เพื่อบอกว่า จุดนี้มีเพื่อไว้เว้นบรรทัดเท่านั้น

ข้อความต่างๆก็สามารถอยู่ได้โดยไม่มี TAG ครอบเช่นกัน แต่เราควรใช้ประโยชน์จาก TAG ต่างๆมากกว่า

นอกจาก TAG สำหรับหัวข้อกับข้อความแล้ว ยังมี TAG ช่วยต่างๆ เช่น TAG strong ที่จะทำให้ข้อความที่ถูกครอบกลายเป็นตัวหนา
